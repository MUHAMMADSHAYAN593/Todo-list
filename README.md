## Usage

### Add Task

1. Click "Add Task" button
2. Enter task title
3. Optionally add details
4. Click "Create"

### Edit Task

1. Click on task
2. Modify description
3. Save changes

### Complete Task

1. Check the checkbox
2. Task marked as complete
3. Appear in completed section

### Delete Task

1. Click delete icon
2. Confirm deletion
3. Task removed

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Add task |
| `Delete` | Delete task |
| `Escape` | Cancel edit |
| `Ctrl+K` | Search |

## Data Storage

- Tasks saved in browser localStorage
- Auto-saves changes
- Persists between sessions
- 5-10MB storage limit
